# coc-tjs-postfix

tjs-postfix coc vim extension

## Install

`:CocInstall coc-tjs-postfix`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-tjs-postfix-keymap)`

## Lists

`:CocList demo_list`
